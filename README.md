# Chrome Dino T-Rex Game
# Описание
Это клон классической браузерной игры "T-Rex Runner", встроенной в Google Chrome. Игра представляет собой бесконечный раннер, где игроку предстоит управлять динозавриком, прыгая через появляющиеся препятствия и собирая бонусы для увеличения счета.

Игру реализовали на Python с использованием графической библиотеки Pygame.

## Установка и запуск
Для начала убедитесь, что у вас установлен Python и пакет `Pygame`. 
Затем выполните следующие команды:

Инструкция по установке
1. Скачать или клонировать репозиторий:
   ```bash
   git clone https://github.com/ila884044-maker/Chrome--REX.git
   cd Chrome--REX
   ```

2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустить игру:
   ```bash
   python main.py
   ```

## Требования
- Python версии 3.6 и выше
- Библиотека Pygame (`pip install pygame`)
